# Start Here

This repository contains an educational and research project studying
emergent coordination and soft role allocation (SRT) using reinforcement
learning in a tactical environment.

If you are opening this project for the first time, **start here**.

---

## What this project is

- A deterministic tactical engine (hex grid, turn-based)
- A Gym-compatible RL environment
- PPO-trained agents (no reward shaping)
- A staged curriculum (P1–P4-B)
- A replay + graphical viewer for visual validation

The focus is on **behavioral emergence**, not game production.
