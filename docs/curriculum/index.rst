Reinforcement Learning Curriculum
=================================

This section documents the **reinforcement learning curriculum**
developed on top of the assault engine.

The curriculum follows a staged design, where each phase introduces
exactly one new source of complexity while keeping all other elements
stable.

The objective is to study how increasingly complex behavior emerges
from structural changes rather than reward shaping.
