Assault‑Env Documentation
=========================

This documentation describes the **Assault‑Env research project**, which
combines a deterministic tactical engine with a reinforcement learning
curriculum to study **emergent coordination and soft role allocation**.

The documentation covers both **system architecture** and
**experimental methodology**, with a strong emphasis on
**reproducibility and offline analysis**.

Scope of the documentation
--------------------------

This documentation includes:

- Tactical engine architecture and rules
- Reinforcement learning environments and curriculum
- Execution and evaluation pipelines
- Offline diagnostics and batch analysis workflows
- Replay generation and visualization

The project is intended for **research and educational use**, not
game production.
