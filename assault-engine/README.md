# Assault Engine

Turn-based tactical engine inspired by the *Assault* wargame system.  
Designed as an independent CORE for simulation, analysis, and future integration
with AI agents and reinforcement learning.

---

## 🎯 Project Objective

The goal of **Assault Engine** is to implement a hex-based tactical game engine that:

- Is **fully independent from AI**
- Faithfully models tactical rules: movement, terrain, combat, and assaults
- Allows deterministic and reproducible simulations
- Provides a solid foundation for:
  - Heuristic AI
  - Decision-making agents
  - Reinforcement learning (RL)

The focus is on **rules and architecture**, not on graphics or UI.

---

## 🧱 Architecture

The project is designed using clearly separated layers:

1. **CORE** (this repository)  
2. Environment (future)  
3. Agents (future)  
4. Orchestration and simulation (future)  
5. Interface and analysis tools (future)

This repository covers **only the CORE layer**.
